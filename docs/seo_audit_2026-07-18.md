## 1. robots.txt

| Fact | Value |
|------|--------|
| Exists in repo | Yes — `site/public/robots.txt` (copied to `dist/` on build) |
| Live URL | `https://www.lisenbart.com/robots.txt` → 301 → `https://lisenbart.com/robots.txt` |
| Live body matches `public/robots.txt` | Yes |

**Full contents (verbatim):**

```
User-agent: *
Allow: /
Disallow: /celana/
Sitemap: https://www.lisenbart.com/sitemap.xml
```

| Directive | Meaning (as written) |
|-----------|----------------------|
| `User-agent: *` | Applies to all crawlers |
| `Allow: /` | Allows indexing of `/` |
| `Disallow: /celana/` | Blocks path `/celana/` |
| `Sitemap:` | Points to `https://www.lisenbart.com/sitemap.xml` |

No other `Disallow` / `Allow` rules present.
